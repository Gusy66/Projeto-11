Desenvolvimento de um site de relacionamento para adultos com o intuito de ajudar as pessoas a se relacionarem melhor.

A segurança do site ainda está em desenvolvimento, aceito qualquer tipo de opinião e ajuda para melhorar tanto o código
quanto a questão da segurança envolvida no aplicativo.

Uma das ideias para assegurar um ambiente seguro, é um cadastro com reconhecimento facial. Dessa forma, o desenvolvimento
desse tipo de ferramenta seria em outra linguagem de programação, como por exemplo: Python.	




Recursos que o aplicativo oferece:

	- Criação de perfis
	- Pesquisa de usuários
	- Chat	
	- Geolocalização 
	- Algoritmo de correspondência
